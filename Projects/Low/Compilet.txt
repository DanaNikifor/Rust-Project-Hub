Компиляция проекта:

```bash
rustc файл.rs
```
после ввода команды вы получите файл с расширением .exe
(Linux)
```bash
./название_файла
```
(Windows)
```bash
название_файла.exe
```

Скачать полный Rust:
Linux:
https://static.rust-lang.org/dist/rust-1.77.2-x86_64-unknown-linux-gnu.tar.xz
Windows:
https://static.rust-lang.org/dist/rust-1.77.2-x86_64-pc-windows-msvc.msi
